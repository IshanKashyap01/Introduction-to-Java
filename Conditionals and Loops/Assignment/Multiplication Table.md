# Multiplication Table

```Java
import java.util.*;

public class Main 
{    
    public static void main(String[] args) 
    {
        // Write your code here
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        for(int i = 1; i <= 10; i++)
        {
            System.out.println(n * i);
        }
    }
}
```
