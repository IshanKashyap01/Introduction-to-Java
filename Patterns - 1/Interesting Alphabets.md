# Interesting Alphabets

```Java
import java.util.*;

public class Solution 
{
    public static void main(String[] args) 
    {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        for(int i = 1; i <= n; i++)
        {
            for(int j = 0; j < i; j++)
            {
                System.out.print((char) ('A' + n - i + j));
            }
            System.out.println();
        }
    }
}
```
